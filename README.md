## Herramientas ciidept
sistema para administrar la Biblioteca Ciidept


```
#!git

git clone https://USER_NAME@bitbucket.org/tecnologiaciidept2/biblioteca2017.git
```


```
#!composer

composer install   
```

```
#!artisan

 php artisan migrate 
```

```
#!artisan

 php artisan db:seed 
```

```
#!artisan

php artisan key:generate
```
se crea el panel
